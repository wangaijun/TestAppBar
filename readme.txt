抽屉布局的优缺点：
优点：在页面的左侧添加抽屉布局之后，用户只需滑动页面的左侧，就可以快速地进入某个页面进行交互，降低了摩擦成本。
缺点：缺点也显而易见，这样做会增加页面模块的复杂度，使页面略显凌乱，导致应用的主题并不突出等。

ActionBar已经绑定到Activity中，是独立于页面布局（ContentView）存在的，与Activity紧密地耦合在一起，难于定制，也无法与其他控件联动，如本例中的AppBarLayout
需要显示或隐藏，使用ActionBar无法实现。这也是多数公司都会使用定制视图代替ActionBar的原因。
为了将导航栏与ActionBar完全解耦，使其重归页面布局的怀抱，MD设计者创建了Toolbar，完全代替ActionBar的功能，通过Toolbar实现多控件之间的联动与动画效果的增强，
为用户带来更加的操作体验。
